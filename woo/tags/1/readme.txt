Enhance WooCommerce checkout with conditional logic, field validation, and field customization.

== Description ==
Billing Process Engineering for Woo helps you streamline your WooCommerce checkout by adding:

Conditional Field Logic: Dynamically adjust checkout fields based on the type of products in the cart (e.g., physical or digital).

Customizable Validation: Allow customers to enter either email or phone for digital products, with validation ensuring accuracy.

Field Customization: Remove unnecessary fields such as company name and order notes to declutter the checkout form.

Easy-to-Use Settings Panel: Manage all plugin features effortlessly from an intuitive settings interface.

This plugin is ideal for businesses looking to simplify their checkout process and improve the user experience.

== Installation ==

Upload the plugin files to the /wp-content/plugins/billing-process-engineering directory, or install the plugin through the WordPress plugins screen directly.

Activate the plugin through the 'Plugins' screen in WordPress.

Go to WooCommerce > Settings > Checkout to configure plugin options.

== Frequently Asked Questions ==

= Can I disable email or phone validation? =
Yes, you can disable this feature from the settings panel under WooCommerce > Settings > Checkout.

= How do I remove unnecessary fields like company name? =
You can enable or disable this feature in the plugin settings.

= Does this plugin support conditional logic? =
Yes, the plugin dynamically adjusts checkout fields based on the type of products in the cart.
